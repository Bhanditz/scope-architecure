client
An application which acts as debugger.
host
An Opera instance to be debugged. The same as debugee.
message
In the context of STP or the scope interface it is an entity in the communication between a host and a client. It is either a command, a response, an event or an error.
service
The meaning depends on the context. A service on the host is a domain specific part of the scope interface. It can be enabled and disabled. So far there is Scope, WindowManager, ConsoleLogger, HTTPLogger, Exec, EcmascriptDebugger and URLPlayer. On the client it means normally an object which exposes the specific interface as a counterpart of the host service as an API.
STP/0
Scope Transfer Protocol version 0. Specifies the format of messages between a host and a client. Used in Opera version 10 and below.
STP/1
Scope Transfer Protocol version 1. Specifies the format of messages between a host and a client. Will be used in future versions of Opera.
JSON
JavaScript Object Notation. Simple markup for data structures.
